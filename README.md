# Sistema de Auditoría Basado en COSO - Formulario HTML

Este proyecto consiste en un formulario HTML para realizar una auditoría basada en el marco COSO. El formulario permite que los usuarios respondan preguntas sobre el cumplimiento de controles internos y riesgos, y proporciona un análisis básico de las respuestas.

## Características

- Formulario interactivo para evaluar los principios del modelo COSO.
- Preguntas estructuradas para identificar incumplimientos.
- Proporciona un análisis básico de los riesgos y controles.

## Tecnologías Utilizadas

- **HTML**: Estructura del formulario.
- **CSS**: Estilo de la página.
- **JavaScript/jQuery**: Validación de formularios y lógica básica.
- **Bootstrap**: Para mejorar la presentación y responsividad.
